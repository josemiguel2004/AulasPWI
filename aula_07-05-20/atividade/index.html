FUNÇÕES DA TEG NO HTML
nome da tag função
!DOCTYPE html informa o navegador o site que está utilizando
HTML Contém todos os elementos de um documento
Head Tags html
Title define o titulo da sua página
Body É nessa tag que os elementos visiveis do seu site devem ser inseridos
p Faz um parágrafo
ol cria uma lista ordenada
ul cria uma lista comum sem ser ordenada
img coloca uma imagem
table cria uma tabela
form cria um formulario
hr cria uma linha
b negrito na versão 4 do html
br quebra linha
do h1 ao h6 define o cabeçalho
link define ligaçoes da pagina com outros arquivos
meta define propriedades da pagina
style define a formataçao em css
script define programaçao de certas funçoes em paginas com script, principalmente com java script

atributos de id,style e class

o ID serve para marcar, idetificar algo, porém ele é unico não pode se repetir em mais de uma maquina

o style permite modificar, alterar o estilo do texto, ou seja, ele alterar cor, tamanho da letra e o tipo da letra

o atributo a class é uma lista de classes de um elemento separada por espaços, permitindo o css e javascript selecionar ou acessar elemntos especificos.
